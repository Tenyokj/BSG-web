# Download now

Install to Applications

Unzip the downloaded file and drag "Battery & System Guardian.app" to your Applications folder.

# Move to Applications folder
mv ~/Downloads/"Battery & System Guardian.app" /Applications/
3

Fix Permission Issues

If macOS marks the app as untrusted (Gatekeeper), run this command in Terminal:

# Fix app permissions
xattr -cr "/Downloads/Battery & System Guardian.app"

Or if you've moved it to Applications:

xattr -cr "/Applications/Battery & System Guardian.app"
4

Launch and Monitor

Launch the app from your Applications folder ‚Äî it will appear as a small icon in your macOS menu bar. Click to view all system stats!

# Open from Terminal (optional)
open /Applications/"Battery & System Guardian.app"

Development & Build

Built with modern technologies for maximum performance, reliability, and a native macOS experience.

# Clone the repository
git clone https://github.com/tenyokj/Battery-System-Guardian-.git
cd battery-guardian

# Install dependencies
npm install

# Run in development mode
npm start

# Build as macOS App
npm run make

The built files will appear inside the out/make/ folder as both .app and .dmg installer.
